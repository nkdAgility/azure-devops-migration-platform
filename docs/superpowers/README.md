# Superpowers Working Documents

This folder is the output location for the `writing-plans` and `brainstorming`
skills (`docs/superpowers/plans/` and `docs/superpowers/specs/`).

Files here are **dated working artefacts** — implementation plans and design
documents produced for a specific piece of work at a specific time. They are
not maintained documentation:

- they are not updated when the system changes
- they may describe designs that were later revised or superseded
- they must not be treated as canonical references

For current documentation, start at [`docs/README.md`](../README.md). For
recorded architectural decisions, see [`docs/adr/`](../adr/README.md).
